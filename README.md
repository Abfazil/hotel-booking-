# HotelEase 🏨

HotelEase is a modern hotel management and booking platform designed to simplify hotel operations and deliver a smooth experience for both guests and administrators.

From reservations and room management to customer experience, HotelEase focuses on clarity, efficiency, and ease of use.

---

## ✨ Features

- 🛏️ Room listing and availability management  
- 📅 Booking and reservation handling  
- 👤 Guest management  
- 🧾 Booking history and records  
- 🔐 Secure authentication and role-based access  
- 📱 Responsive design for desktop and mobile  

---

## 🚀 Tech Stack

- **Frontend:** HTML / CSS  
- **Backend:** Node.js / Express 
- **Database:**  MySQL
- **Authentication:** JWT / OAuth  
- **Hosting:** AWS   

---

🚧 **In active development**
